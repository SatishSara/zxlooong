Band Object Implementation Sample "BandObjs"

BandObjs is a simple example that demonstrates the implementation of the four types of band objects. These are Desk Bands, Explorer Bars Communication Bands and Tool Bands. See the Band Objects topic in the Microsoft Platform SDK documentation for more information on band objects.

Once the sample is built, it can register itself. If you use Visual C++ 6.0 to build this project, it will automatically register the server when the server is built. The included makefile will also attempt to register the server. If you need to register the server manually, just execute the following from the command line, substituting the path where the DLL is located for "<path>":

regsvr32.exe /s /c <path>\BandObjs.dll
