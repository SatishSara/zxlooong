File and directory distribution of GlobalDev application:
====================================

.\ - Main project of Globaldev.exe
-----------------------------------------

Source files:
	gentab.c
	global.c
	nlsapi.c
	textapi.c
Header files:
	global.h
	resource.h
	res.h (copy of .\gres\resource.h)

Resource files:
	Global.rc
	icon1.ico
VS created files:
	Global.dsp
	Global.dsw
	Global.ncb
	Global.opt
	Global.plg
Readme files:
	global.txt - to be present on the same share as global.exe (used in the "general" tab of the application).
	readme.txt - this file

-------------------------------------------------------------------

	gres409.c

	resource.h (renamed to res.h in .\ directory)
